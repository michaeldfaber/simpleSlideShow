Download simpleSlideShow.js

A lightweight and customizable jQuery slide show plugin.


Get Started

Include simpleSlideShow.js and jQuery.js files in your document. Use jQuery selectors to call simpleSlideShow() method on an empty div element and quickly create a slick image rotation.


Function Parameters

time:
The duration each photo is shown in seconds. Decimals are allowed. Does not accept values less than or equal to zero. Defaults to 8 seconds.
transition:
The duration of the transition animation in tenths of a second. Decimals are allowed. Does not accept values below zero. To disable the transition, set to 0. Defaults to 1 second.
urls:
An string[] which holds the image sources to be looped through.

Styling

Images inherit size and position from their parent div element. Images also maintain their aspect ratio. You can override this and directly style the img elements by using the simpleSlideShowElement css class. 

<style>
    [Your Slide Show Div] {
        //changes
    }
    [Your Slide Show Div].simpleSlideShowElement {
        //changes
    }
</style>

Demo

The bare bones:

Making use of every default parameter, this is all you need to get a slide show started. 

<div id="slideShow"></div>

<script>
    $('#slideShow').simpleSlideShow({
        urls: 
        [
            "img src 01",
            "img src 02",
            "img src 03"
        ]
    });
</script>

Get Fancy

Add your own styles, classes, and values. 

<style>
    #slideShow {
        margin: 0px;
        padding: 0px;
        position: absolute;
        height: 540px;
        width: 960px;
    }
    #slideShow.simpleSlideShowElement {
        border-radius: 10px;
        opacity: 80%;
    }
</style>
